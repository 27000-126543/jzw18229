## 1. 产品概述

创意市集（CreativeMart）是一个数字素材与创意模板销售平台，连接创作者与买家。创作者可上传 Figma 模板、PPT 模板、字体、图标包、代码片段、Notion 模板等数字作品，设置售价与授权类型（个人用途/商用授权）；买家通过分类浏览、关键词搜索和标签筛选找到所需素材，购买后获取下载凭证与授权信息。平台提供创作者后台数据看板、月度结算、免费作品关注者体系、评分留言等完整交易生态。

- 目标用户：设计师、开发者、内容创作者、营销人员等需要数字素材的职场人群
- 核心价值：为创作者提供变现渠道，为买家提供高质量可信赖的数字素材来源

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 买家 | 邮箱/手机号注册 | 浏览、搜索、购买、下载、评分留言、关注创作者 |
| 创作者 | 邮箱/手机号注册 + 创作者认证 | 上传作品、设置售价/授权、查看数据看板、提现结算 |
| 游客 | 无需注册 | 浏览分类目录、查看作品预览（低清） |

### 2.2 功能模块

1. **首页**：Hero 展示区、分类导航、热门推荐、新上架作品、免费精选、创作者推荐
2. **浏览/搜索页**：关键词搜索、分类筛选、标签过滤、排序方式、作品网格列表
3. **作品详情页**：高清预览轮播、作品描述、授权选项、价格、创作者信息、相关作品、评分留言
4. **创作者后台**：数据看板（销量/收入/下载趋势）、作品管理、结算记录、关注者管理
5. **上传/发布页**：作品信息填写、文件上传、预览图上传、价格与授权设置
6. **购物车/结算页**：订单确认、支付方式、授权确认、下载凭证生成
7. **我的购买页**：已购作品列表、下载凭证、授权信息查看
8. **个人中心**：基本信息、关注列表、消息通知

## 3. 核心流程

### 3.1 买家购买流程
买家进入平台 → 浏览/搜索作品 → 查看作品详情与高清预览 → 选择授权类型 → 加入购物车/立即购买 → 确认订单 → 支付 → 生成下载凭证与授权信息 → 下载作品 → 评分留言

### 3.2 创作者发布流程
创作者认证 → 进入上传页 → 填写作品信息 → 上传源文件与预览图 → 设置价格与授权 → 提交审核 → 审核通过上架 → 关注者收到推送

### 3.3 结算流程
系统每月1日统计上月数据 → 计算总收入 → 扣除平台手续费（15%）→ 生成结算单 → 创作者确认 → 打款到创作者账户

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深炭黑 (#0D0D0D) + 亮金橙 (#FF6B2C) 作为强调色，辅以暖灰 (#2A2A2A) 和象牙白 (#F5F0EB)
- **按钮风格**：圆角矩形（8px），主按钮金橙渐变填充，次按钮描边风格，hover 带微妙上浮阴影
- **字体**：标题使用 "Playfair Display" 衬线体营造高端创意感，正文使用 "DM Sans" 无衬线体保证可读性
- **布局风格**：桌面端顶部固定导航 + 侧边筛选面板 + 内容区卡片网格，大量留白突出作品视觉
- **图标风格**：线性图标，2px 描边，圆角端点，与整体设计协调
- **整体调性**：暗色高端创意市场风格，类似 Behance/Dribbble 的专业感，但更偏商业交易

### 4.3 响应式设计

- 桌面优先设计，断点：1440px / 1024px / 768px / 480px
- 平板端：筛选面板收起为顶部筛选条，卡片网格从4列变2列
- 移动端：底部Tab导航替代顶部导航，卡片单列展示，购买面板底部固定

### 4.4 动效设计

- 页面加载：元素依次淡入上浮，stagger 间隔 80ms
- 卡片交互：hover 上浮 4px + 阴影扩散，点击微缩放反馈
- 滚动触发：作品卡片进入视口时淡入动画
- 搜索建议：下拉面板滑入动画
- 购买完成：成功弹窗带粒子庆祝效果
- 数据看板：数字计数动画，折线图绘制动画
